1. Giới thiệu về HTML Forms

Form trong HTML là một phần tử được sử dụng để thu thập dữ liệu từ người dùng. Form bao gồm các phần tử như input, button, checkbox, radio button, select, textarea, v.v.

Cú pháp cơ bản của một form:

Code Example:

<form action="/submit" method="POST">
    <!-- Các phần tử form ở đây -->
</form>

Các thuộc tính quan trọng của form:

  • action: URL nơi dữ liệu form sẽ được gửi đến
  • method: Phương thức HTTP (GET hoặc POST)

2. Form Cơ Bản

Hãy bắt đầu với một form đơn giản bao gồm input text, password và button.

Code Example:

<form>
    <label for="username">Tên đăng nhập:</label>
    <input type="text" id="username" name="username">
    
    <label for="password">Mật khẩu:</label>
    <input type="password" id="password" name="password">
    
    <button type="submit">Đăng nhập</button>
</form>

3. Các Loại Input (Input Types)

HTML5 cung cấp nhiều loại input khác nhau để phù hợp với từng loại dữ liệu.

Code Example:

<form>
    <!-- Email -->
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    
    <!-- Number -->
    <label for="age">Tuổi:</label>
    <input type="number" id="age" name="age" min="1" max="120">
    
    <!-- Date -->
    <label for="birthday">Ngày sinh:</label>
    <input type="date" id="birthday" name="birthday">
    
    <!-- Color -->
    <label for="color">Màu yêu thích:</label>
    <input type="color" id="color" name="color">
    
    <!-- Range -->
    <label for="volume">Âm lượng:</label>
    <input type="range" id="volume" name="volume" min="0" max="100">
</form>

4. Radio Button & Checkbox

Radio button cho phép chọn một tùy chọn duy nhất, trong khi checkbox cho phép chọn nhiều tùy chọn.

Code Example:

<form>
    <!-- Radio Buttons -->
    <p>Giới tính:</p>
    <input type="radio" id="male" name="gender" value="male">
    <label for="male">Nam</label>
    
    <input type="radio" id="female" name="gender" value="female">
    <label for="female">Nữ</label>
    
    <input type="radio" id="other" name="gender" value="other">
    <label for="other">Khác</label>
    
    <!-- Checkboxes -->
    <p>Sở thích:</p>
    <input type="checkbox" id="music" name="hobby" value="music">
    <label for="music">Âm nhạc</label>
    
    <input type="checkbox" id="sports" name="hobby" value="sports">
    <label for="sports">Thể thao</label>
    
    <input type="checkbox" id="reading" name="hobby" value="reading">
    <label for="reading">Đọc sách</label>
</form>

5. Select (Dropdown) & Textarea

Select tạo menu dropdown, textarea cho phép nhập văn bản nhiều dòng.

Code Example:

<form>
    <!-- Select/Dropdown -->
    <label for="city">Thành phố:</label>
    <select id="city" name="city">
        <option value="">-- Chọn thành phố --</option>
        <option value="hanoi">Hà Nội</option>
        <option value="hcm">TP. Hồ Chí Minh</option>
        <option value="danang">Đà Nẵng</option>
        <option value="cantho">Cần Thơ</option>
    </select>
    
    <!-- Textarea -->
    <label for="message">Tin nhắn:</label>
    <textarea id="message" name="message" rows="4" cols="50"></textarea>
</form>

6. Các Thuộc Tính Quan Trọng (Attributes)

Các thuộc tính giúp kiểm soát và validate dữ liệu form.

Code Example:

<form>
    <!-- Required: Bắt buộc phải nhập -->
    <label for="fullname">Họ tên (bắt buộc):</label>
    <input type="text" id="fullname" name="fullname" required>
    
    <!-- Placeholder: Văn bản gợi ý -->
    <label for="phone">Số điện thoại:</label>
    <input type="tel" id="phone" name="phone" placeholder="0123456789">
    
    <!-- Disabled: Vô hiệu hóa -->
    <label for="country">Quốc gia:</label>
    <input type="text" id="country" name="country" value="Việt Nam" disabled>
    
    <!-- Readonly: Chỉ đọc -->
    <label for="userid">User ID:</label>
    <input type="text" id="userid" name="userid" value="USER12345" readonly>
    
    <!-- Maxlength: Giới hạn độ dài -->
    <label for="code">Mã code (tối đa 6 ký tự):</label>
    <input type="text" id="code" name="code" maxlength="6">
</form>

Demo:

Giải thích các thuộc tính:

  • required: Trường bắt buộc phải điền
  • placeholder: Hiển thị văn bản gợi ý
  • disabled: Vô hiệu hóa input (không gửi dữ liệu)
  • readonly: Chỉ đọc (vẫn gửi dữ liệu)
  • maxlength: Giới hạn số ký tự tối đa

7. Ví Dụ Tổng Hợp - Form Đăng Ký

Kết hợp tất cả các kiến thức đã học để tạo một form đăng ký hoàn chỉnh.

Code Example:

<form action="/register" method="POST">
    <h3>Form Đăng Ký Tài Khoản</h3>
    
    <label for="reg-username">Tên đăng nhập:</label>
    <input type="text" id="reg-username" name="username" required>
    
    <label for="reg-email">Email:</label>
    <input type="email" id="reg-email" name="email" required>
    
    <label for="reg-password">Mật khẩu:</label>
    <input type="password" id="reg-password" name="password" required>
    
    <label for="reg-birthday">Ngày sinh:</label>
    <input type="date" id="reg-birthday" name="birthday">
    
    <p>Giới tính:</p>
    <input type="radio" id="reg-male" name="gender" value="male">
    <label for="reg-male">Nam</label>
    <input type="radio" id="reg-female" name="gender" value="female">
    <label for="reg-female">Nữ</label>
    
    <label for="reg-country">Quốc gia:</label>
    <select id="reg-country" name="country">
        <option value="vn">Việt Nam</option>
        <option value="us">United States</option>
        <option value="jp">Japan</option>
    </select>
    
    <input type="checkbox" id="reg-terms" name="terms" required>
    <label for="reg-terms">Tôi đồng ý với điều khoản</label>
    
    <button type="submit">Đăng ký</button>
    <button type="reset">Xóa form</button>
</form>
